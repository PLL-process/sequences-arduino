# Les 20 visuels pédagogiques intégrés

| Nº | Visuel | Usage principal |
|---:|---|---|
| 1 | Situation déclenchante | Comprendre le problème d’arrosage |
| 2 | Prototype final attendu | Visualiser le résultat du projet |
| 3 | Cycle de l’eau | Relier technologie, environnement et SVT |
| 4 | Besoin et contraintes | Lire le cahier des charges |
| 5 | Composants du système | Identifier Arduino et modules Grove |
| 6 | Architecture fonctionnelle | Suivre la mesure jusqu’à l’action |
| 7 | Chaîne d’information | Acquérir, traiter, communiquer |
| 8 | Chaîne d’énergie | Alimenter, distribuer, convertir, transmettre, agir |
| 9 | Entrées, traitement, sortie | Distinguer A0/A1/A2 et D6 |
| 10 | Capteur d’humidité A0 | Réaliser le premier montage |
| 11 | Capteur de lumière A1 | Ajouter une mesure de luminosité |
| 12 | Capteur de niveau A2 | Protéger la pompe |
| 13 | Relais D6 | Commander la pompe |
| 14 | Montage final complet | Vérifier l’intégration |
| 15 | Erreur de branchement | Prévenir une alimentation dangereuse |
| 16 | Signal analogique | Comprendre une grandeur continue |
| 17 | Signal logique | Comprendre LOW et HIGH |
| 18 | Mesure à action | Relier seuil, décision et commande |
| 19 | Algorigramme | Préparer l’algorithme |
| 20 | Code final annoté | Identifier les fonctions dans le programme |

Les deux premiers visuels apparaissent après la situation déclenchante. Une sélection adaptée apparaît dans chaque mission. La collection complète est disponible dans une bibliothèque repliable située avant le tableau de bord.
