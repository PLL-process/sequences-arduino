# Bibliothèque visuelle TechnoQuest

Le fichier `technoquest-visuels.svg` contient vingt illustrations vectorielles originales utilisées par le projet HTML.

- format : SVG réutilisable et redimensionnable sans perte ;
- intégration : balises `<svg><use>` générées par `visuals.js` ;
- accessibilité : chaque visuel possède un texte alternatif et une légende ;
- licence pédagogique : CC BY 4.0.

Les visuels couvrent la situation déclenchante, le prototype final, les composants Arduino/Grove, les chaînes d’information et d’énergie, les branchements A0/A1/A2/D6, les signaux, l’algorigramme et le code annoté.
